<h1>Map and Sets</h1>

<p>In this chapter 9 we are going to talk about <strong>Map</strong> and <strong>Sets</strong>.</p>

<p>
<strong>Map:</strong> is a collection of key-value pairs. Unlike objects, in MAP, the keys can be of any kind - even objects or functions! In addition, it maintains the order of insertion, which can be mega useful in some situations.
</p>

<p>
<strong>Set:</strong> is like an array but each value is unique. it's very usefull when we want to set unique values in array as ID's, phone numbers, e-mail (as we know the e-mail is unique amd different for everyone).
</p>
