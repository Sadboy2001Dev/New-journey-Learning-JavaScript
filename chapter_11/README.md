<h1>Iterators</h1>

<p>
ES6 introduces two very important new concepts: iterators and generators. Generators depend on iterators.
</p>

<p>
An iterator is roughly analogous to a bookmark: it helps you keep track of where you are. An array is an example of an iterable object: it contains multiple things (like pages in a book), and can give you an iterator (which is like a bookmark).
</p>
